# userscript-douban-site-227778
豆瓣深圳租房团增强脚本

## 为什么会有这个东西
* 租金那么贵，中介那么多，找到一个不用中介费的地方真不容易
* 小站没有搜索功能，各种信息混合在一起，看起来太费事
* 豆瓣字体太小，还把标题隐藏掉一部分，总是要一个一个点开进去看，太费时
* 不知道怎么放到userscript上去，就先放这了
* ~~github上还没有东西，怪不得会纠结中介费~~

## 用处
* 屏蔽掉广告之类的与小站主题无关的东西
* 把豆瓣蛋疼的长标题省略掉的后半部分重新显示出来
* 过滤掉一些不感兴趣的主题，只是根据标题来过滤

## 使用方法
* chrome:原生支持用户脚本，打开chrome://extensions/,把文件拖放进去即可进入安装程序
* firefox:安装greaseMonkey扩展，然后打开about:addons，切换到用户脚本，然后把文件拖进去即可
* 其他:理论上来说只要支持用户脚本的就都可以（没有测试过）

## 其他一些看起来没什么用的TIPS
* 理论上来说可以在修改@include和body使用的ID之后在任意豆瓣小站使用
* 这个理论可以扩展到豆瓣小组
